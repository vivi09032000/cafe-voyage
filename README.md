# ☕ Cafe Voyage

咖啡廳探索 App — 串接 [Cafe Nomad API](https://cafenomad.tw/developers/docs/v1.2) 真實資料

## 功能
- 依城市瀏覽咖啡廳
- 快速篩選：不限時 / 插座多 / WiFi 穩
- 工作友善綜合排行
- 收藏功能
- 詳細頁：評分、營業時間、官網連結

## 本地開發

```bash
npm install
npm run dev
```

## 部署到 Vercel

```bash
npm install -g vercel
vercel
```

## 技術
- React 18 + Vite
- Cafe Nomad API v1.2
- 無後端，純前端串接
